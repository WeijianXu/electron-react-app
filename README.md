# electron-react-app

## Commonds
---
```bash
npm run start           # start up the project
npm run build:          # build the project
npm run test            # test the project
npm run electron        # run the app with electron in development model 
npm run electron:url    # run the app with electron from a url
npm run pack            # pack the app in the dist direction with electron-builder 
npm run dist            # build the app and then pack the app into the installation packages with electron-builder
npm run postinstall     # check the deps with electron-builder before you pack the app
```
